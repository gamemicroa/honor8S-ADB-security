# honor8S-ADB-security

Easy switch from Google apps to alternatives for better privacy and control.

---

##  What this project does
This tool helps you:
- Remove/disable selected Google apps using ADB
- Install privacy-friendly alternatives like Firefox
- Debloat your device without root access
- Improve privacy and reduce Google dependency

---

## Warning
**Use at your own risk.**

Removing system apps may cause instability, bugs, or require a factory reset.

Always make a backup before running any scripts.

---

## Requirements
- Windows PC (ADB installed)
- USB cable
- USB debugging enabled (Developer Options)
- Honor 8S device connected

---

## Features
- One-click debloat scripts (ADB based)
- Google app removal (safe user-level uninstall)
- Alternative app installation
- Lightweight and fast setup

---

##  Example app replacements

| Google App        | Alternative(s)                          |
|------------------|------------------------------------------|
| Chrome           | Firefox / Brave                          |
| Gmail            | K-9 Mail / Proton Mail                   |
| Google Maps      | Organic Maps / OsmAnd                    |
| Google Photos    | Aves Gallery / Ente Photos               |
| Google Drive     | Proton Drive                             |
| Google Search    | DuckDuckGo / Brave Search                |
| Google Play      | F-Droid                                  |

---

## Usage

1. Enable USB debugging on your phone  
2. Connect phone to PC  
3. Run the script:

```bat
debloat.bat
```

## Goal
Make Android cleaner, faster, and more privacy-focused without root.
